## プライバシーポリシー (Privacy Policy)

本プライバシーポリシーは、スマートフォン向けアプリ「otocla」（以下、「本アプリ」といいます）の提供において、ユーザーの個人情報およびユーザーデータがどのように収集、利用、および管理されるかについて定めるものです。

本アプリをご利用いただいた場合、本プライバシーポリシーに同意したものとみなします。

### 1. 収集する情報と利用目的
本アプリは、ユーザーの利便性向上のため、以下の情報を取得・利用する場合がありますが、開発者が個人を特定できる情報（名前、住所、電話番号、メールアドレス等）を無断で収集することはありません。

#### 1-1. アプリ内で入力されたデータ
本アプリ内でユーザーが入力したデータ（計算内容、商品の価格、容量、履歴、メモなど）は、ユーザーのデバイス本体（ローカルストレージ）にのみ保存されます。これらのデータが開発者のサーバーに送信されたり、第三者に共有されたりすることはありません。

#### 1-2. お問い合わせ・フィードバック
本アプリ内の「機能要望・バグ報告（フィードバック）」フォームから送信された内容は、アプリの改善および不具合修正の目的でのみ利用します。送信内容はGoogle Forms等の外部サービスを介して開発者に送信されますが、ユーザーが任意で個人情報を入力しない限り、個人を特定することはありません。

### 2. 第三者サービスによる情報収集（広告の配信について）
本アプリでは、無料版の運営・維持のために第三者配信の広告サービス（Google AdMob）を利用しています。
広告配信事業者は、ユーザーの興味に応じたパーソナライズ広告を表示するために、Cookieやデバイスの識別情報（Appleが提供する広告識別子（IDFA）など）、匿名化されたアクセス情報を使用することがあります。

これらの情報は、広告配信事業者のプライバシーポリシーに従って管理されます。詳細については、以下のリンクをご確認ください。
*   [Google ポリシーと規約（広告）](https://policies.google.com/technologies/ads?hl=ja)

ユーザーは、お使いの端末（iOS）の「設定」>「プライバシーとセキュリティ」>「トラッキング」または「Appleからの広告」の設定より、パーソナライズ広告の制限や、トラッキングの拒否を行うことが可能です。

### 3. アプリ内課金（プレミアムプラン）について
本アプリでは、広告の非表示などを提供するアプリ内課金（サブスクリプションおよび買い切り型プラン）を提供しています。
課金処理および決済情報はすべてApple Inc.（App Store）を通じて安全に行われます。

開発者はユーザーのクレジットカード情報や詳細な決済情報を取得・保持することはありません。購入履歴の復元や有効性の確認（StoreKit）を行うために、匿名化された購入レシート情報のみを利用します。
